# 3️⃣ Authentication Flow (Your Current Setup)

### Files Involved:

* context/AuthContext.jsx
* hooks/useAuth.js
* routes/ProtectedRoute.jsx
* api/axios.js

### Flow:

1. User logs in → Login.jsx
2. auth.js calls backend → receives JWT
3. Token stored (localStorage or context)
4. AuthContext stores:

   * user info
   * role
   * token
5. ProtectedRoute checks:

   * Is user logged in?
   * Does user role match route?
6. axios.js attaches token automatically to API requests
7. Backend validates token again

Important:
Frontend role check = UI protection
Backend role check = Real security
